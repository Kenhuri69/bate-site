# bate-site

Vitrine publique (GitHub Pages) + distribution de l'APK pour **BATE**, un RPG
Android fan-made basé sur *The Beginning After The End* de TurtleMe.

- **Site** : déployé via GitHub Pages (workflow `.github/workflows/pages.yml`).
- **APK** : publié dans les [Releases](../../releases) de ce dépôt (asset `BATE.apk`).

Le **code source du jeu** est hébergé dans un dépôt privé séparé. Ce dépôt ne
contient que la page vitrine et les binaires distribués.
